# PointBreak Client Side
Exchange application to get real time values from Binance

## DEMO
[Visit PointBreak App](https://pointbreak-dashboard-cs0wrq7fe-dslaurindo.vercel.app/auth/signin)

### Technologies
* NextJS
* Chakra-UI
* Typescript
* Next-Auth
* SWR
* Websocket
* GitHub Actions
* Vercel

#### Installation

`yarn install`

#### Running

`yarn dev`

http://localhost:3000
